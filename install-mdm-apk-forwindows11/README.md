# Headwind MDM Android 安装工具 (Windows 11)

本目录提供在 Windows 11 上批量安装 Headwind MDM Android 应用的一键脚本。

## 安装位置

```
D:\
└── install-mdm-apk-forwindows11\     ← 解压到此目录
    ├── README.md                        # 本说明文档
    ├── install-mdm.bat                  # 一键安装脚本 (CMD版)
    ├── install-mdm.ps1                  # 一键安装脚本 (PowerShell版，美化)
    ├── hmdm-android-6.45-release.apk  # MDM 主应用安装包
    ├── v4.0.0-prod.apk                # Pro 版本安装包
    └── adb-windows\                     # Windows 版 ADB 工具
        ├── adb.exe                      # ADB 主程序
        ├── AdbWinApi.dll                # ADB Windows API
        ├── AdbWinUsbApi.dll              # ADB USB 驱动
        └── ...                          # 其他 ADB 组件
```

> **注意：** 脚本使用相对路径引用同目录下的 APK 和 ADB 文件，**请勿移动文件位置**，保持 `D:\install-mdm-apk-forwindows11\` 目录结构完整。

## 前置要求

### 1. 平板端设置

在开始安装前，请确保平板已开启 **USB 调试模式**：

1. 进入平板 `设置` → `关于平板电脑`
2. 连续点击 `版本号` 7 次，开启开发者模式
3. 返回 `设置` → `系统` → `开发者选项`
4. 开启 `USB 调试`
5. （可选）开启 `允许 USB 安装`

### 2. Windows 端设置

- Windows 11 系统
- USB 数据线（推荐使用原装线）
- 运行脚本前**先解压所有文件**，确保目录结构完整

## 使用步骤

### 快速开始

1. 将 `install-mdm-apk-forwindows11` 文件夹解压到 **D 盘根目录**
2. 确认目录结构如上所示，所有文件都在原位
3. 用 USB 线连接第一台 Android 平板
4. **右键** `install-mdm.bat` → **以管理员身份运行**
   - 或使用 PowerShell 版：`install-mdm.ps1`（美化界面）
5. 等待脚本自动完成所有操作
6. 根据平板提示授予设备管理员权限
7. 完成后拔掉平板，连接下一台
8. 脚本会自动检测并开始安装下一台

### 运行脚本的方式

| 方式 | 操作 | 说明 |
|------|------|------|
| CMD 版 | 右键 `install-mdm.bat` → 以管理员身份运行 | 经典批处理，兼容性好 |
| PowerShell 版 | 右键 `install-mdm.ps1` → 用 PowerShell 运行 | 美化界面，带动画和进度条 |

## 平板端权限说明

安装过程中，平板会弹出以下权限请求，请**全部允许**：

| 权限类型 | 说明 |
|---------|------|
| 设备管理员 | 必须同意，否则 MDM 功能无法使用 |
| 存储权限 | 用于访问 SD 卡和文件管理 |
| 电话权限 | 用于获取 IMEI 和设备信息 |
| 位置权限 | 用于 GPS 位置追踪功能 |
| 相机权限 | 用于远程控制和照片上传 |
| 自启动 | 确保 MDM 服务开机自启 |
| 默认桌面 | 必须设为默认启动器才能使用 Kiosk 模式 |

## 常见问题

### Q: 平板无法被检测到

**解决方法：**
1. 更换 USB 数据线（部分线只支持充电）
2. 在平板上重新插拔 USB 线
3. 在平板上点击"允许 USB 调试"弹窗
4. 安装平板的 USB 驱动（可在平板官网下载）

### Q: 提示"设备所有者设置失败"

**解决方法：**
1. 在平板上手动设置：设置 → 安全 → 设备管理器
2. 找到 "Headwind MDM" → 激活为设备管理员
3. 返回脚本按任意键继续

### Q: 如何设置默认桌面启动器？

**解决方法：**
1. 安装完成后，平板会弹出"选择默认桌面"对话框
2. 选择 "Headwind MDM" 并勾选"始终使用"
3. 如果没有弹窗，手动设置：设置 → 应用 → 默认应用 → 桌面 → 选择 Headwind MDM

### Q: 安装失败，提示签名不匹配

**说明：** 本 APK 使用自定义签名，请勿卸载已安装的旧版 MDM，否则需要先清除数据。

### Q: 如何批量安装多台设备？

**操作方法：**
1. 在脚本运行完后，拔掉当前平板
2. 连接下一台平板
3. 脚本会自动检测到新设备并开始安装
4. 重复以上步骤直到所有设备安装完成

### Q: 如何确认安装成功？

**检查方法：**
1. 平板桌面出现 Headwind MDM 图标
2. 设置 → 安全 → 设备管理器中可见 Headwind MDM
3. 打开 MDM 应用，显示设备已注册
4. 平板重启后自动进入 MDM 桌面（Kiosk 模式）
